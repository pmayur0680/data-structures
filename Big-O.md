## Singly Linked Lists

- Insertion O(1)
- Removal O(1) beginning or O(N) end
- Searching O(N)
- Access O(N)

## Doubly Linked Lists

- Insertion O(1)
- Removal O(1)
- Searching O(N)
- Access O(N)

## Stacks

- Insertion O(1)
- Removal O(1)
- Searching O(N)
- Access O(N)

## Queue

- Insertion O(1)
- Removal O(1)
- Searching O(N)
- Access O(N)

## BST

- Insertion O(log n)
- Searching O(log n)

## Binary Heaps

- Insertion O(log N)
- Removal O(log N)
- Searching O(N)
